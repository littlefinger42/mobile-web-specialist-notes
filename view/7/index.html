Overview

var is function scoped. let and const are block scoped (curley brackets). const is immutable, let isn't.

Template literals are string literals allowing embedded expressions. You can use multi-line strings and string interpolation features with them. A string literal should be inside two back tickets ``. The expression should be in ${...}

Previously you had to do:
console.log('Fifteen is ' + (a + b) + ' and\nnot ' + (2 * a + b) + '.');
Now you can do:
console.log(`Fifteen is ${a + b} and not ${2 * a + b}.`);
